# Christats

Este repositorio público compila código y libretas jupiter en R y python para diferentes análisis estadísticos.

Su objetivo es autodidáctico.

La mayor parte del código está escrito en ingles, aunque los comentarios estén en español (no siempre es el caso.)


Las implementaciones con pymc y bambi utilizarán blackjax or nutpie ya que el sampler de fábrica genera errores en windows.
